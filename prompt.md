# Задача: Личный сайт-резюме (лендинг)

Создай одностраничный современный сайт-резюме на чистом HTML/CSS/JS 
(без фреймворков, без сборщиков — статика, которая будет хоститься на GitHub Pages).

## Стиль и требования к дизайну
- Современный, чистый, минималистичный дизайн. НЕ нужен "вау-эффект",
  кричащие анимации, тяжёлые градиенты или парящие 3D-элементы.
- Сдержанная деловая эстетика: подходит для государственного/корпоративного 
  IT-специалиста (DevOps/DBA), а не для креативного портфолио дизайнера.
- Тёмная и светлая тема (переключатель) — по желанию, но приветствуется.
- Аккуратная типографика (например, Inter / IBM Plex Sans / system-ui),
  умеренные отступы, читаемая сетка, лёгкие микроанимации при скролле
  (fade-in), без избыточности.
- Адаптивная вёрстка (mobile-first).
- Один длинный скролл со sticky-навигацией по секциям (или простое меню сверху).
